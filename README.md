# Iw-SistemaCursos
# 🎓 Sistema de Cursos (Java)

Projeto desenvolvido com o objetivo de praticar conceitos fundamentais de programação orientada a objetos em Java.

## 📌 Sobre o projeto

O sistema simula a gestão de uma instituição de ensino, permitindo o cadastro e organização de:

* 👨‍🎓 Alunos
* 👩‍🏫 Professores
* 👨‍💼 Funcionários
* 📚 Cursos
* 📍 Endereços
* 📞 Contatos

Além disso, utiliza enums para controle de status cadastral.

## 🧠 Conceitos aplicados

* Programação Orientada a Objetos (POO)
* Herança e Polimorfismo
* Encapsulamento
* Organização em pacotes
* Uso de Enums
* Separação de responsabilidades (camadas)

## 🗂️ Estrutura do projeto

```
src/
 ├── aplicacao
 ├── entidade
 ├── servico
```

## ▶️ Como executar

1. Clone o repositório:

```
git clone https://github.com/seu-usuario/seu-repo.git
```

2. Compile o projeto:

```
javac aplicacao/Programa.java
```

3. Execute:

```
java aplicacao.Programa
```
